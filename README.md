# Allen Abraham — Portfolio

Personal portfolio site. Experience, projects, and a bit about me.

**Live site:** [allenabraham106.github.io/Allen_Portfolio](https://allenabraham106.github.io/Allen_Portfolio/)

## Tech

- React 19
- Vite 7
- React Router (HashRouter)
- Framer Motion

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output is in `dist/`.

## Deploy

The repo uses GitHub Actions to build and push `dist/` to the `gh-pages` branch. Push to `main` to trigger a deploy. In **Settings → Pages**, set the source to the `gh-pages` branch (root).
